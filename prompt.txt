🧾 Project Title

Simple Expense & Profit Tracker SaaS for Small Hotels & Restaurants

🌍 Project Overview

Build a production-ready SaaS web application that enables small hotels and restaurants to easily track:

Daily expenses

Daily sales

Cash vs Bank balances

Automatically generated Profit & Loss (P&L) reports

The system must be simple, fast, mobile-friendly, and built for non-technical business owners who need clear financial visibility without accounting complexity.

This must be a real working SaaS, not a UI mockup.

🎯 Core Objectives

Accurate daily financial tracking

Clear separation of Cash vs Bank transactions

Real-time balance updates

Automatic Profit & Loss calculations

Clean dashboards and visual reports

Secure multi-tenant SaaS architecture

Scalable backend structure for future growth

🧱 Mandatory Tech Stack
Layer	Technology
Frontend	React (Next.js App Router and tailwind CSS)
Backend	Node.js + Express.js
Database	PostgreSQL (Preferred) or MongoDB
ORM	Sequelize (Postgres) or Mongoose (MongoDB)
Authentication	JWT (Access + Refresh token flow)
UI Styling	Tailwind CSS
Charts	Recharts or Chart.js
Payments	Stripe (for SaaS subscription billing)
🧩 Core Features & Functional Requirements
1️⃣ Authentication & Business Setup

User Registration & Login

JWT-based authentication

Password hashing using bcrypt

Each user belongs to one business

Business types: HOTEL or RESTAURANT

Single role for MVP: OWNER

Middleware to protect all business routes

Multi-tenant isolation (users only access their own business data)

2️⃣ Daily Expense Management

Users can:

Add daily expenses with:

Date

Category: Food, Staff, Electricity, Rent, Maintenance, Other

Amount

Payment Method: CASH or BANK

Optional notes

Edit and delete expenses

Filter by:

Date range

Category

Payment method

👉 When an expense is added:

Deduct from Cash balance if CASH

Deduct from Bank balance if BANK

3️⃣ Sales Management

Users can:

Add daily sales entries:

Date

Amount

Payment Method: CASH or BANK

Allow multiple entries per day

Auto-calculate total daily sales

👉 When a sale is added:

Increase Cash balance if CASH

Increase Bank balance if BANK

4️⃣ Cash vs Bank Balance Tracking

System must maintain real-time running balances:

Cash Balance

Bank Balance

Rules:

Balances auto-update after every sale or expense

Prevent negative cash balance (validation error)

Show balances clearly on dashboard

5️⃣ Profit & Loss (P&L) Reporting

Auto-generated financial report including:

Total Sales

Total Expenses

Net Profit / Loss

Filters:

Today

This Week

This Month

Custom Date Range

Visualizations:

Sales vs Expenses (Bar/Line chart)

Expense category breakdown (Pie chart)

Cash vs Bank distribution

💳 6️⃣ Stripe Subscription Billing (SaaS Feature)

Integrate Stripe Checkout for subscriptions

Plans:

Free Trial (optional)

Basic Monthly Plan

Features:

Only active subscribers can access dashboard

Webhook to update subscription status

Middleware to restrict access for inactive users
